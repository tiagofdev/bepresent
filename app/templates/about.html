About this website

Functionalities

  • Create Students and Classes records
  • Take attendance for each class day and generate reports
  • Attendance Reports available for download in CSV format
  • Browse your classes through the calendar interface
  • Sort students by first and last name
  • Class creation algorithm does not permit overlapping classes at the same time and day
  • Does not permit creating multiple classes with same name
  • Quick Add Students when creating classes

Techonologies used

  • Python
  • Flask / Jinja
  • MySQL
  • HTML / CSS / JS
  • Foundation
  • Google Cloud Platform

Security Features

  • Authentication System
  • Password Security using Hashing with Salt
  • Multiple failed login attempts prevented
  • Timed Session Control
  • Automatic logout after inactivity
  • Only one login per browser
